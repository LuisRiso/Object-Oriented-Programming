# Object-Oriented Programming

This repository contains class materials, exercises, assignments, practical implementations, and study projects developed during the **Object-Oriented Programming** course at the **Federal University of Itajubá (UNIFEI)**.

The course focuses on the fundamentals of object-oriented software development, combining programming practice in Python with software modeling concepts such as UML diagrams, use cases, class diagrams, sequence diagrams, encapsulation, inheritance, polymorphism, exception handling, object persistence, and the MVC architectural pattern.

## 📚 About the Course

Object-Oriented Programming is a course designed to introduce the main principles of object-oriented software development.

Throughout the course, the activities progress from an introduction to Python to the modeling and implementation of object-oriented systems. The repository includes exercises, examples, assignments, exam practice, UML modeling activities, graphical interface experiments, and implementations involving classes, objects, relationships, inheritance, polymorphism, exceptions, persistence, and MVC.

The main purpose of this repository is to document the learning process and organize the practical applications developed during the discipline.

## 🎯 Learning Objectives

The main objectives of this repository are to:

* Practice object-oriented programming using Python.
* Understand the concepts of classes, objects, attributes, and methods.
* Apply encapsulation and information hiding.
* Work with constructors, method overriding, inheritance, and polymorphism.
* Model software using UML diagrams.
* Develop use case, class, and sequence diagrams.
* Understand relationships between objects, such as association, aggregation, and composition.
* Use abstract classes and interfaces.
* Handle exceptions in object-oriented applications.
* Work with collections of objects.
* Implement object persistence.
* Explore the MVC architectural pattern.
* Practice graphical interface concepts using Tkinter.

## 🏗️ Object-Oriented Programming Concepts

Object-Oriented Programming is based on organizing software around objects that combine data and behavior.

In this repository, the main OOP principles are explored through Python implementations and modeling exercises:

* **Abstraction:** representing only the essential characteristics of a problem.
* **Encapsulation:** protecting and organizing data inside classes.
* **Inheritance:** creating specialized classes from more general ones.
* **Polymorphism:** allowing different objects to respond to the same operation in different ways.
* **Composition and Aggregation:** modeling relationships between objects.
* **Persistence:** saving and retrieving object data.
* **MVC:** separating responsibilities between model, view, and controller layers.
